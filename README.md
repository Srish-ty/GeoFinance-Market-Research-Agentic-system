# GeoFinance Market Research Agent

GeoFinance Market Research Agent is an ADK-based AI agent system that analyzes
global macro events and market instruments to help users with market research.

## Features

- Macro event analysis
- Sector impact mapping
- Market snapshot analysis
- Multi-agent orchestration using ADK

## Stack

- Google ADK
- Gemini
- Python
- Cloud Run

## Run locally

```bash
adk web
```

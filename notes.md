## Workflow for Market Intelligence Agent

```
User Query
   ↓
Orchestrator Agent
   ├── Macro Intelligence Agent
   │      ├── News Search Tool
   │      ├── Geopolitical Summary Tool
   │      └── Sector Impact Mapper
   │
   └── Market Instrument Agent
          ├── Stock/Fund Data Tool
          ├── Trend Analyzer
          └── Comparison Tool

   ↓
Final Synthesis
   - key event
   - impacted sectors
   - relevant stocks/ETFs/MFs
   - metrics snapshot
   - risks and watchpoints
```
